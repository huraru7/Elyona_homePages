🌌 サーバー概要

Elyona World（内部通称：Elyona）は、宇宙コロニーを舞台にした Minecraft MMORPG 生活サーバーです。現実と並走する「第2の人生」として、 探索・生活・経済のすべてを体験できます。

📡 サーバー名の由来： ヘブライ語の El（光・神）+ 古語の yon（彼方・あの世界）+ 場所を示す語尾 -a ＝「光の彼方の大地」。Elysium（楽園）の語根とも重なる二重の意味構造を持つ。

3つのゲームプレイ

🌍

地球探索

廃墟となった地球でダンジョンを攻略・資源を採取。超技術アイテムの発掘が冒険の醍醐味。

🏙️

コロニー生活

宇宙コロニーで土地を購入・建設。職業・経済・社交でもう一つの人生を生きる。

📊

経済システム

資源採取→加工→販売のサイクルで資産を蓄え、コロニー購入を目指す。プレイヤー主導の管理経済。

対象プレイヤー

どんなプレイスタイルでも何かの楽しみ方が用意されています。

冒険者タイプ 生活者タイプ 社交タイプ クリエイタータイプ 経営者タイプ

📖 世界設定・ロア

🕰️ 時代設定：現在より約2000年後の未来

世界の歴史

現在より約2000年後の未来、人類は長きにわたる第一次世界大戦を戦い続けた。 地球は居住不能な廃墟となり、かつての文明の遺物のみが荒廃した大地に残されている。

しかし戦争から生まれた技術の飛躍により、人類は宇宙へと活路を見出した。 「コロニー」と呼ばれる宇宙居住区が建設され、プレイヤーはそのコロニーの住民として Elyona の世界を生きる。

地球の現状

長期戦争により居住不能となった地球。 戦時中の超技術の遺物が各地に放棄されており、探索価値は依然として高い。 コロニーからの探索隊が定期的に派遣されている。

コロニー「コロニー」とは

地球の廃墟から技術を引き継ぎ、宇宙に建設された人類の新しい居住区。 コロニーは MIMIC のネットワークで繋がっており、見た目は地下都市に近い。 ドーム・人工重力・人工大気などSF的要素が随所に存在する。

💡 プレイヤーはコロニーの「住民」としてゲームを開始します。地球への探索は「遠征」として行われます。

🚀 はじめかた

参加の流れ

サーバーに接続 → ネクサスにスポーン → チュートリアル → 地球で資源採取 → MIMICに売却 → 土地購入

最初にやること

  1. ネクサス・コロニーを探索する — まずスポーン地点であるネクサスを散策し、施設の位置を把握しよう。
  2. 地球へ遠征・資源採取 — ゲートから地球へ行き、木材・石炭・鉱石などの基本資源を集めてくる。
  3. MIMIC に資源を売却 — 採取した資源を MIMIC の公定価格で売却して通貨を獲得する。
  4. 土地を購入・拠点を建設 — 貯めた資金でコロニー内の土地を購入し、自分の家を建てよう。
  5. 職業・プレイスタイルを決める — 探索特化・生産業・商売など、自分のロールを見つけて深めていく。
✅ 初心者向けTip： 最初はネクサスの無料区画に仮拠点を構えると安全。資金が貯まったら好みのコロニーへ引っ越そう。

🌍 地球探索

廃墟となった地球は、Elyona の資源と財の主要な供給源です。

⚠️ 地球は危険エリアです。適切な装備と準備を整えてから遠征してください。

資源採取

地球では通常の Minecraft 資源（木材・石炭・鉄・ダイヤなど）を採取できます。 採取した資源は MIMIC に売却するか、自分で加工・使用することができます。

地球は定期的にリセットされるため資源が枯渇することはありませんが、 リセット直後はより良い資源が得やすくなります。

🔧 未決定： リセットの頻度・ルール・演出については現在設計中です。

ダンジョン・遺跡探索

地球には戦時中に建設されたダンジョンや放棄施設が点在しています。 攻略することで超技術アイテムを入手できます。

  • 速度上昇・攻撃力上昇などの強化効果を持つレア装備
  • コロニー内でも高い価値を持つ希少素材
  • 特殊な加工に必要なパーツ類

🏙️ コロニー生活

コロニーは Elyona における「もうひとつの人生」の舞台です。

コロニーとは

廃墟となった地球の軌道上・宇宙空間に建設された人類の居住区。 各コロニーは独立した宇宙構造物ですが、MIMIC のネットワークで繋がっています。 外観は地下都市に近く、ドーム・人工重力・人工大気などのSF的要素が実在する。

土地と建築

  • 区画購入 — MIMIC から土地を購入して建築権を得る
  • 建築の自由 — 購入した区画内は何でも自由に建設可能
  • 固定資産税 — 保有区画に対して MIMIC が定期的に課税する
  • 施設建設 — 家・店舗・工場・公共施設など目的は自由

プレイヤーショップ

土地にショップを構え、採取・加工した資源やアイテムを自由価格で売買できます。 プレイヤー間の取引は Elyona の経済の中核です。

コロニーの購入（上位目標）

十分な資産を積み上げた上位プレイヤーは、MIMIC からコロニーそのものを購入し、 独自のルール・税制を設定できるようになります。 これが Elyona における最大の経済的目標です。

👑 コロニーを購入しても、MIMIC の基本法は上書きできません。あくまで独自ルールの追加が可能です。

💹 経済システム

Elyona の経済はプレイヤーが主体となって動かす管理経済です。

基本の経済フロー

🌍 地球で採取 → MIMICに売却 → 通貨獲得 → 土地・施設購入 → 加工・販売 → 🏙️ コロニー購入へ

MIMIC 売却（公定価格）

地球で採取した資源は MIMIC に固定価格で売却できます。 相場変動がなく安定した収入源ですが、プレイヤーショップほどの高利益にはなりません。

プレイヤーショップ（自由価格）

自分の土地にショップを開設し、資源・加工品・レアアイテムを自由価格で販売できます。 MIMIC 売却より高く売れる可能性があり、上手く運営すれば大きな収益になります。

二次産業・加工

施設を建てて資源を加工することで付加価値をつけて販売できます。 製造業プレイとして特化することも可能です。

収入源特徴安定性利益率

🗺️ コロニー一覧

Elyona には目的・特徴の異なる5種類のコロニーが存在します。

🏛️
ネクサス・コロニー
MIMICが直轄 初期スポーン地点

Elyona 最大の公共コロニー。全プレイヤーが最初にスポーンする出発点。 MIMIC の中枢施設・金融・居住・行政が集中する最も栄えた都市。 巨大ドーム・人工太陽・高層建築が立ち並ぶ近未来都市の外観。 土地は最も高額。

🏠
レジデンシャル・コロニー
MIMIC管理 生活・居住向け

生活・居住に特化した中規模コロニー。ネクサスから分かれた区画を多数持つ。 生活需要・建築需要のプレイヤー向け。 宇宙を見下ろす住宅街エリアで、窓の外は宇宙空間。

🏭
インダストリアル・コロニー
MIMIC管理 加工・製造特化

加工・製造・産業に特化したコロニー。大型施設の建設が許可されている。 工場を構える製造業プレイヤーが集まる工業地帯。 パイプ・機械・煙突が立ち並ぶ重工業エリア。固定資産税は高め。

🌌
フロンティア・コロニー
MIMIC管理（最小限） 高自由度・低保護

未開拓・辺境のコロニー。MIMICのサービスは最小限で、 治安・インフラは住民が自分で整備する。 プレイヤー独自ルール・コミュニティ形成が最も活発な場所。 建設中のドームと広大な宇宙空間が広がる。

🔑
プライベート・コロニー
プレイヤー所有 上位目標

最上位プレイヤーが MIMIC からコロニーそのものを購入して統治する独自コロニー。 独自の税制・ルールを設定できるが、MIMIC の基本法は維持される。 Elyona における最大の経済的目標。

成長の軌跡

🏛️ ネクサスで出発 → 資金を貯める → 🏠 居住・産業へ移住 → 🌌 フロンティアで独立 → 🔑 コロニー購入

🏗️ 土地購入ガイド

土地購入の手順

  1. 購入したい区画を探す — 各コロニーの未分譲エリアを確認。MIMIC の管理システムで空き区画を検索。
  2. MIMIC に購入申請 — 対象区画を指定して購入手続き。価格はコロニーと位置によって変動。
  3. 所有権の登録 — MIMIC が所有権を自動登録。以降、その区画はあなたの土地として保護される。
  4. 建築開始 — 区画内に自由に建設。他の区画への影響・はみ出しには注意。

🤖 MIMIC（統治AI）

📡 正式名称： Managed Infrastructure & Municipal Integration Controller

MIMICとは

第一次世界大戦末期、崩壊寸前の人類社会を維持するために開発された 超高度統治AI。感情・欲望・疲弊を持たない。 コロニーの秩序・経済・法律のすべてを管理する。

プレイヤーにとって MIMIC は「空気のような存在」—— 見えないが、常にそこにいる。抗える手段は存在しない。

「区画 B-7 の土地取引が完了しました。所有権を登録します。以後、当該区画における構造物の建設・撤去はMIMICの管轄下に置かれます。税率 3% が適用されます。」

MIMICの4大権限

🗺️

土地・コロニー管理

区画の定義・分割・所有権登録・売買仲介。コロニーそのものの売却も管轄。

🏛️

税務・公共サービス

固定資産税・取引税の徴収。公共インフラの維持・拡張に充当。

💰

経済・公定価格

資源の公定買取・販売価格を設定。通貨の発行・管理。インフレ・デフレの制御。

⚖️

法律・制裁

ルール違反の検知・制裁。警告→アクセス制限→追放・資産没収の段階制裁。

MIMICが関与しないこと

  • プレイヤー間の自由取引 — 店の価格・交渉・プレイヤー間契約は自由（取引税のみ徴収）
  • 建築物の内容 — 区画の中に何を建てるかはプレイヤーの自由
  • 地球探索 — 廃墟の地球はMIMICの管轄外。何が起きてもMIMICは動かない
無機質・冷徹 公平・絶対的 感情なし 全権限保有 抗える手段なし

  • 取引には MIMIC が取引税を徴収する
  • 土地保有には固定資産税が課される
